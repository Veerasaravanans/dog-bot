L293D Setup Instructions

Critical Setup Steps

1. Remove ENABLE Jumpers: The L293D module ships with jumpers on the ENABLE A and ENABLE B pins. These jumpers tie the ENABLE pins to 5V (full speed, no PWM control). You MUST physically remove both jumpers before connecting GPIO 2 and GPIO 4.

2. Verify Connections:

  • GPIO 2 (ESP32-CAM) → ENABLE A (L293D) - Orange wire
  • GPIO 4 (ESP32-CAM) → ENABLE B (L293D) - Light blue wire
  • Both ENABLE jumpers removed from L293D board

3. Test PWM Control: After uploading the firmware, test motor speed by sending commands like /motor?dir=forward&speed=100 (slow) and /motor?dir=forward&speed=255 (full speed). If motors only run at full speed regardless of the speed parameter, the jumpers are still installed.
